# Multi-Agent Interview Coach

## Setup
```bash
uv venv
source .venv/bin/activate
uv sync
```

## Run CLI
```bash
uv run python -m interview_coach.cli
```

## Run Scenario
```bash
uv run python -m interview_coach.scenarios --scenario examples/scenarios/sample.json
```

## Run Tests
```bash
uv run pytest
```

## Output logs
```bash
ls runs/
```
